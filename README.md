## 🌟 Why Novacraft?

Novacraft is a passion project inspired by the original vision for Minecraft, before its takeover by corporate interests. We aim to create a voxel-based game that remains true to Notch's ideals while addressing some of Minecraft's most significant shortcomings. Here's what makes Novacraft different:

- **Community-Centric**: We believe in a development process that's 100% driven by players.
- **Unified Experience**: Our Java/Bedrock-like codebase, powered by [TeaVM](https://teavm.org/), ensures seamless web compatibility.
- **Freedom for Modders**: With open-source tools, modders have unprecedented creative control.
- **Fair Pricing**: Just a one-time purchase of $16.99, with no hidden fees or exploitative practices.

## 🚀 Core Features

### 🎮 Gameplay
- 🏭 **Industrial Revolution Era**: Build factories and harness steam power.
- 🐉 **Tameable Red Dragons**: Befriend once-feared creatures.
- 🌌 **Expanded Dimensions**: Explore new worlds and revive classics.
- 💧 **Survival 2.0**: Experience a more immersive survival mode.

### 🔧 Technical
- ⚡ **Browser-Based Play**: Enjoy Novacraft on any device with over 900 FPS optimization.
- 🧱 **Robust Codebase**: Java-like modding and C++ performance.
- 📦 **Nova World Format**: Infinite chunks, backwards compatibility.
- 🛡️ **AI Moderation**: Safe, clean, and friendly online community.

### 🌍 Community
- 🗳️ **Vote on Updates**: Influence the future of Novacraft via Discord polls.
- 🧪 **Modding API Access**: Early tools for developers.
- 📜 **Transparent Policies**: Co-developed with legal experts.

## 👥 How to Contribute
1. **Developers**: Contribute to our open-source codebase:
	* **Optimize TeaVM**: Help us optimize our Java→JS compiler for better performance.
	* **WebAssembly Support**: Assist in implementing wasm support for faster performance.
	* **Fix Bugs**: Help squash bugs and improve overall stability.
2. **Artists**: Create stunning steampunk textures for the game:
	* **Textures**: Create textures for the game's blocks, items, and other visual assets under the CC-BY 4.0 license.
3. **Testers**: Put our web client to the test on various devices:
	* **Chromebooks**: Help us test the game on ChromeOS devices.
	* **Other Devices**: Test the game on other devices you have access to.
