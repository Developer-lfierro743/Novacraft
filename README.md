# 🛠️ About Novacraft

Built by Novaforge Studios and crafted by @Developer-lfierro743 (main developer/creator), Novacraft pushes the limits of sandbox games. We're not here to play nice—we’re here to break boundaries and deliver what others won’t.

# What makes Novacraft different?

🏭 Industrial Revolution Vibes – Embrace the grind with advanced machinery, complex automation, and cutting-edge tech.

🌍 Limitless World – Procedurally generated with Perlin + Simplex noise—no boring barriers here!

🧱 True Creativity – Ray tracing, infinite build height (yes, infinite), and tech so futuristic it’ll make your GPU sweat.

🔓 Open Modding API – We won’t lock you out. Build it, break it, share it. Just play nice with the rules.

🕹️ Cross-Platform Dream – PC? ✅ Android? ✅ Web? Coming soon. Consoles? Maybe...

You bet. Anything Notch or Mojang ditched? We’ll make it work—and better. Infinite build height? Done. Sky Dimension? You got it.
