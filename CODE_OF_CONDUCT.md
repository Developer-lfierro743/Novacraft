Novacraft Code of Conduct

1. Purpose

Novacraft is a hybrid (closed-source/open-source) project, meaning that while the core game remains closed-source, we provide open-source components (e.g., modding API, custom assets) for community contribution.

This Code of Conduct ensures a welcoming and respectful environment for all contributors, whether they are reporting issues, suggesting features, or contributing to open-source aspects of the project.

--
2. Scope

This Code of Conduct applies to:

Contributions to the open-source components of Novacraft.

Discussions in official Novacraft GitHub issues, pull requests, Discord, and community forums.

Any public interaction related to Novacraft, including user-generated content, mods, and community discussions.

--
3. Community Guidelines

We expect all contributors to follow these principles:

✅ Be Respectful

Treat others with kindness and respect.

Avoid hate speech, discrimination, or harassment based on race, gender, sexual orientation, disability, religion, or any other characteristic.


✅ Keep Discussions Productive

Provide constructive feedback when reporting issues or reviewing contributions.

Stay on topic when discussing features, bug reports, or modding support.


✅ Respect Licensing & Ownership

Do not decompile, reverse-engineer, or redistribute the closed-source parts of Novacraft without explicit permission.

Open-source components (such as the modding API) follow separate licensing terms—read and adhere to those licenses.


✅ Contribution Guidelines

If contributing to the open-source parts (e.g., modding API), follow the coding standards and documentation provided.

Respect the hybrid license—modifications should be for open-source components only unless explicitly permitted.

Do not upload or distribute copyrighted content without proper authorization.
--

❌ Prohibited Behavior

No unauthorized forks of the private core repository.

No impersonation or false claims regarding affiliation with the Novacraft team.

No spamming, self-promotion, or irrelevant content in discussions.

No cheating, hacking, or exploits that disrupt fair gameplay.

--
4. Enforcement

Violations of this Code of Conduct may result in:

A warning from moderators or project maintainers.

Removal of comments, pull requests, or issues.

Temporary or permanent bans from Novacraft’s GitHub and community spaces.

--
5. Reporting Issues

If you encounter behavior that violates this Code of Conduct, please report it via:

GitHub Issues (for technical concerns).

Community Moderators (for social interactions in Discord/forums).


Reports will be reviewed confidentially and handled appropriately.

--
6. Acknowledgment & Updates

By contributing to Novacraft (whether through mods, discussions, or feature requests), you agree to follow this Code of Conduct.

The Novacraft team reserves the right to update this Code of Conduct as needed.


TL;DR:

Be respectful & constructive.

Do not fork, redistribute, or modify the closed-source parts.

Feel free to contribute to open-source components (mods, assets, API).

Violations can lead to removal or bans.
